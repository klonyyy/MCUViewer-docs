# General Settings

General settings window is used to control application settings. Open by selecting `Options->General` from the top menu.

```{figure} ./images/GeneralSettings.png
:width: 1000px
:align: center
```

## 1. General section

- `Project auto save` - select if you'd like to automatically and periodically save your project
- `Log directory` - shows the current log directory. It can be accessed quickly by pressing `Open` button next to it. 
- `Float precision` - change the number of decimal places when displayin floats in the workspace.  

## 2. Plot export

The Plot Export section provides options for saving the current plot area as a *.png file. Plots can be saved either by navigating to File -> Save plots to .png or by using the keyboard shortcut `Ctrl+P`.

By default, all plots are saved to the system’s Pictures directory. Users with an enterprise license have the option to change the save destination, or enable the `Save as` option when making the screenshot.

## 3. License section

License section allows to open the license file of MCUViewer. License files have *.lic extension and have to be purchased from [MCUViewer website](https://mcuviewer.com). 

```{note}
Before purchasing online seat-based license please test the server connection using the `Test` button. 
```

### Offline license activation

1. Select the *.lic license file
2. Make sure the status shows `VALID` 

```{figure} ./images/GeneralSettingsOfflineLicense.png
:width: 1000px
:align: center
```

### Online License 

1. Select the *.lic license file
2. Make sure the status shows `VALID` 
3. Select `Take` button. After taking a seat the seat can be released for another user after a cooldown period. 

```{figure} ./images/GeneralSettingsOnlineLicense.png
:width: 1000px
:align: center
```
